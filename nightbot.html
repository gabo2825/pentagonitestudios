Nightbot-specific .jsons

  • large arrays
  • eyes.json: an array for a bunch of eye emotes we use. Array's name is "eyes"
  • gordonramsay.json: 3 separate arrays "gr1", "gr2", "gr3" corresponding to @radravioli_'s insult generator
  • LMS.json: [CONTAINS EXPLETIVES] an array with the transcript of the popular "lemme smash" meme. Array's name is "LMS"
  • "functions"
  • gaussian.json: creates a variable "gaus" with a normal distribution from (0,1) //originally made by joshuakcockrell on stackoverflow, adapted by gem
  • typo_0.json: requires an input string "text" that will then be output with a random number of typos. Original iteration that swaps and duplicates some letters
  • specific commands
  • fish_0.json: The original fish code! (with some added functionality to work with the current command) just change fish_1.json to fish_0.json to use the legacy code
  • fish_1.json: 2021, Feb 08 update: fish are now generated based on a normal distribution. Use this as the contents of the nightbot command
  • fish_off.json: change fish_[x].json (x being version number) to fish_off.json to disable fishing without deleting the command
  • for testing only
  • test.json: this is just a single json with a const array "texttest"
  • gaboemote.json: a long array of emotes Gabo sent me. array's name is "emotes"
